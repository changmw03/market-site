# TrendMart Modern Ecommerce

Figma Make의 Modern Ecommerce Website 구조를 참고해 만든 정적 쇼핑몰 웹사이트입니다.

## 주요 기능

- 홈, 상품, 인기딜, 결제 독립 화면 전환
- 총 36개 상품 데이터
- 카테고리별 6개 상품 구성
- 상품 검색
- 인기 검색어 칩
- 카테고리 필터
- 브랜드 필터
- 가격대 필터
- 추천순, 낮은 가격순, 높은 가격순, 평점순 정렬
- 재고 있음 필터
- 인기딜 전용 화면
- 상품 상세 모달
- 별점 클릭 리뷰 모음 이동
- 리뷰 작성과 로컬 저장
- 최근 본 상품
- 상품 비교 최대 3개
- 가격 알림 토글
- 찜 목록 필터
- 장바구니 담기, 수량 변경, 삭제
- 장바구니 전체 비우기
- 할인상품 원가/상품 할인액 반영
- 쿠폰 코드 `SAVE10` 10% 할인
- 일반 배송, 빠른 배송, 매장 픽업 배송 옵션
- 결제 요약
- 주문 완료 후 주문 내역 저장
- 로컬 저장 기반 장바구니/찜 유지
- 중복 렌더 방지와 lazy 이미지 로딩 기반 최적화

## 실행

`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.
